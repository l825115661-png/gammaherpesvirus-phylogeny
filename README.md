# gammaherpesvirus-phylogeny

MSc Bioinformatics dissertation project at the University of Glasgow.

## Workflow

VMR → NCBI Protein → MMseqs2 → Homology Groups → MAFFT → VeryFastTree
Protein families analysed:

- Major Capsid Protein
- DNA Polymerase
- RNR Large Subunit
- Glycoprotein B
- Glycoprotein H
- BGLF5¥

## Dataset

42 gammaherpesvirus genomes obtained from ICTV VMR and NCBI Protein.

## Software

- MMseqs2
- MAFFT
- VeryFastTree
- Biopython
